#Sudoku - Web Version
==================

This is my fifth week project at Makers Academy. 
It's a website that will allow you to play sudoku online.
![alt tag](https://github.com/stefaniacardenas/sudoku/blob/master/public/sudoku.png)

* Sinatra
* HTML5
* CSS3
* Fistface
* S3
* Heroku
* NewRelic
* Shotgun
